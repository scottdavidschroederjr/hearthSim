Hearthstone Set Completion Simulator
A tool to estimate how many packs you would need to open to complete a card set based on how many cards you already own from the set by running 3000 rounds of pack opening simulations.







Retrieve Your Information
To save yourself the trouble of having to count how many of each rarity of cards you own for each set, you can use HSReplay's collection tracker to run a personalized simulation.

  1. Download and install Hearthstone Deck Tracker
  2. Sign in on the Deck Tracker App
  3. Open your collection in game with the Deck Tracker active to sync
  4. View collection info by set on HSReplay's Site
