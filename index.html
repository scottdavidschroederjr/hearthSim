Hearthstone Set Completion Simulator
A tool to estimate how many packs you would need to open to complete a card set based on how many cards you already own from the set by running 10000 rounds of pack opening simulations.







Retrieve Your Information
To save yourself the trouble of having to count how many of each rarity of cards you own for each set, you can use HSReplay's collection tracker to run a personalized simulation.

  1. Download and install Hearthstone Deck Tracker
  2. Sign in on the Deck Tracker App
  3. Open your collection in game with the Deck Tracker active to sync
  4. View collection info by set on HSReplay's Site



The Cost of Hearthstone
With the lack of free methods or events for players to gain packs to counteract Hearthstone's unforgiving pack RNG, it is incredibly costly for players to complete their collection.

Determining what a "competitively viable completion percentage" for each set in standard would be a complete project in of itself. But even using an incredibly conservative of estimate of 10% standard set completion rate, a player would on average need to spend more than $200 on card packs play Hearthstone at a competitive level.


However, these calculations do not take into consideration free rewards that are offered to players which have been at the center of discourse as of late. But even if reworked rewards system was able to slash the cost in half, a new player is being expected to spend about the cost of two AAA games to find out if they like playing Hearthstone.



Potential Solutions
Simulating how Blizzard could dramatically improve the Hearthstone player experience by just slightly adjusting a few variables.

As I was putting the final touches on this project, I ran across this thread over on the Hearthstone subreddit that recommended adjusting the pity timers to make obtaining cards easier for players. The simulation allows for tweaks like this to be made so ran it with a few adjustments suggested from the thread:

  1. Double the speed of the legendary and epic pity timers
  2. Disenchanting cards giving double the current amount of dust
  3. Cards cost half as much to craft & disenchanting giving double dust
  4. All above adjustments done together
